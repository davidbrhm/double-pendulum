# Double Pendulum

A minimalist double pendulum simulation and chaos parameter map explorer written in pure C using Raylib.

## 🎯 Goals

- **Learn C properly:** Master low-level memory management, pointers, and software architecture without relying on heavy frameworks.
- **CPU-bound rendering:** Keep the math and drawing strictly on the CPU, making it as optimized as possible through multi-threading.
- **Make it look epic:** Transition from a simple physics simulation into a high-performance visual art piece, including 3D chaos clouds and procedural audio.

## 🛠️ Tech Stack & Platform

- **Language:** C (C11)
- **Graphics Library:** Raylib
- **Build System:** CMake
- **Dependency Management:** Homebrew (`Brewfile` included)

> ⚠️ **Platform Note:** This project is developed and tested exclusively on macOS (Apple Silicon). While C and Raylib are cross-platform by nature, I have no idea if or how this runs on Windows or Linux at the moment.
